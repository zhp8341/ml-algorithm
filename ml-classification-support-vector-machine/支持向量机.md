## 一、什么是支持向量机

~~~~
支持向量机（Support Vector Machine，SVM）是一种经典的监督学习算法，广泛应用于分类、回归和异常检测等问题上。
SVM 核心思想是通过构建最大间隔超平面对数据进行分类，其中“最大间隔”表示两个类别之间的空隙被最大化，而“超平面”则表示一个 n-1 维的线性决策边界，即在 n 维空间中分割两类的线性平面。对于非线性可分的情况，通过使用核函数将数据映射到高维空间，在高维空间中寻找最优超平面来解决问题。
~~~~

## 二、svm有点和缺点

~~~~
1、优点
    1. 在高维空间中有效地处理线性不可分问题。
    2. 可以有效地处理高维数据。
    3. 可以通过选择合适的核函数来适应各种类型的数据。
    4. 对于数据量较小的情况下，使用 SVM 模型可达到比较优秀的结果。
2、缺点
    1. 数据量较大时，训练时间较慢。
    2. 对噪声和缺失数据较为敏感。
    3. 需要对参数进行调优，选择合适的参数比较困难。
    
总之，SVM 是一种强大的机器学习方法，可以以最大间隔的方式来进行分类，对于数据量较小的情况下，使用 SVM 模型可达到比较优秀的结果。但在使用 SVM 时需要注意模型的缺陷和参数的选择。
~~~~


## 三、其他说明

~~~~
Scikit-learn 库中的 SVM 实现是使用了 libsvm 库，具有多个可用的参数。以下是其中一些常用的参数：

C: 正则化参数，对误分类的惩罚力度。C 越大，误分类的惩罚越严格，容易出现过拟合的情况。

kernel: 内核函数的类型，包括线性核函数、多项式核函数和高斯核函数等。

degree: 多项式核函数的次数。

gamma: 高斯核函数的宽度参数，控制样本点之间的“相似度”。

coef0: 核函数中的截距项。

shrinking: 是否使用收缩启发式算法来加速训练，默认为 True。

tol: 收敛容忍度，在训练结束后的目标函数值达到此值时停止迭代。

cache_size: 内存缓存大小，以 MB 为单位，可设置为较大值以加速计算。

更详细的参数说明可以参考 Scikit-learn 官方文档中 SVM 的相关部分。根据不同的数据特点和业务场景，我们需要选择适当的参数进行调整和优化，以获得更好的模型性能。
~~~~
